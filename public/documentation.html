Achievement 2 Project Brief

This web application will provide users with access to information about different movies, directors and genres. Users will be able to sign up, update their personal ifnormation, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET NONE A JSON object holding data about all the movies
Get the data about a single movie, by title /movies/[Title] GET None A JSON object holding data about a single movie, containing an ID, title, year, genre and directors. Example: { title: 'Black Panther', year: '2018', genre: [ { slot: 1, genre: 'fantasy', }, { slot: 2, genre: 'action', }, { slot: 3, genre: 'adventure', }, ], directors: 'Ryan Coogler', }
Add a movie to list of movies /movies POST A JSON object holding data about the movie to add, structured like: { title: 'The Dark Knight', year: '2008', genre: [ { slot: 1, genre: 'fantasy', }, { slot: 2, genre: 'action', }, { slot: 3, genre: 'adventure', }, ], directors: 'Cristopher Nolan', }, A JSON object holding data about the movie that was added, including and ID: { title: 'The Dark Knight', year: '2008', genre: [ { slot: 1, genre: 'fantasy', }, { slot: 2, genre: 'action', }, { slot: 3, genre: 'adventure', }, ], directors: 'Cristopher Nolan', },
Delete a movie by ID /movies/[ID] DELETE None A text message indicating whether the student was successfully removed
Update a movie title in a certain year /movies/[Title]/[Year] PUT None A text message indicating the movie, title and year that was updated
GET the directors of a movie /movies/[Title]/[director] GET None A text message indicating the director of the movie
